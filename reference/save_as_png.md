# Save a PNG screenshot of a htmlwidget

The `save_as_png` function takes a screenshot of a htmlwidget as a PNG
image.

## Usage

``` r
save_as_png(widget, filepath, background = "white", delay = 0.5)
```

## Arguments

- widget:

  a htmlwidget

- filepath:

  a filepath where to save the file

- background:

  the background color underneath/behind the htmlwidget

- delay:

  a delay (in seconds) to wait before taking the screenshot
